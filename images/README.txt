BEACH HOUSE IMAGE FOLDERS
=========================

To add images to the comparison page:

1. Download/save images from VRBO for each property
2. Name your images using this pattern:

   POOL IMAGES:
   - pool.jpg (or .png, .webp) - First pool image
   - pool2.jpg                  - Second pool image
   - pool3.jpg                  - Third pool image
   - ... and so on

   EXTERIOR IMAGES:
   - exterior.jpg              - First exterior image
   - exterior2.jpg             - Second exterior image
   - exterior3.jpg             - Third exterior image
   - ... and so on

3. Place them in the corresponding folder:

   images/
   ├── option-1/    ← Spacious Beach House (VRBO #4146676)
   │   ├── pool.png
   │   ├── pool2.png
   │   ├── exterior.jpg
   │   └── exterior2.jpg
   ├── option-2/    ← Large Luxurious Home (VRBO #2873463)
   ├── option-3/    ← Family Retreat (VRBO #3737974)
   ├── option-4/    ← Galveston Gem - BEST VALUE (VRBO #2156016)
   ├── option-5/    ← Charming Classic (VRBO #3142857)
   ├── option-6/    ← Beachside Comfort (VRBO #3252017)
   ├── option-7/    ← Family-Friendly (VRBO #1269804)
   ├── option-8/    ← Island Time - TOP RATED (VRBO #3284616)
   ├── option-9/    ← Sandpiper House (VRBO #4379912)
   ├── option-10/   ← 2 Pools + Volleyball (VRBO #2757575)
   └── option-11/   ← The Oasis - TOP RATED (VRBO #3975201)

4. Refresh the page - images will automatically appear!

GALLERY FEATURES:
- Left/Right arrows appear when you hover over images
- Click arrows to cycle through multiple images
- Image counter shows "1 / 3" etc.
- Keyboard: Hover over image and use Left/Right arrow keys

SUPPORTED FILE FORMATS:
- .jpg, .jpeg, .png, .webp, .gif

HOW TO GET IMAGES FROM VRBO:
1. Click any "View on VRBO" link on the comparison page
2. Right-click on the image you want
3. Select "Save image as..."
4. Save as pool.png, pool2.png, exterior.png, exterior2.png, etc.

NOTE: The page will show colorful placeholders if no image file is found.
